# rc505mk2-decoder
Upload your .RC0 files from your RC-505 MK2 and take a look at your settings lol



Drag-and-drop viewer for rc-505 mk2 memory files (.RC0). Its basically .xml so it decodes it and makes it readable.

**[Try here](https://abduloops.github.io/rc505mk2-decoder/)**

You can also just download 'index.html' and run it locally. Doesnt need to install any dependencies or nothing

## How to use ts

1. Back up any RC-505 mk2 memory to a USB compatible device (compoter👁️👁️) (Menu button x3 -> USB -> Right, then Storage Mode ON).
2. Find a 'MEMORYXXX.RC0', XXX being 3 digits, from ROLAND -> DATA folder.
3. Drag and drop into the box
4. bingo bango bongo
